────────────────────────────────────────────────────────────────────────────────
A. TRIGGER SOUND LIBRARY
────────────────────────────────────────────────────────────────────────────────

PURPOSE
A library of recordings organized by SUDs level, with 3–5 distinct
recordings per trigger type per band. Variation within bands is
essential — the brain habituates to a single recording and stops
generating a genuine response. Different recordings of the same trigger
(different people, different contexts, different acoustic qualities)
each require fresh processing.

STEP 1 — MAP YOUR HIERARCHY FIRST
Before sourcing any sounds, complete the trigger hierarchy mapping from
Section 4I of the protocol. For each trigger category (e.g. eating sounds,
mouth breathing, sniffing, repetitive tapping), assign a rough SUDs range
and note context sensitivity (specific people, proximity, etc.). This map
is your procurement list.

STEP 2 — FOLDER STRUCTURE
Create the following folder structure before downloading anything:

  /cc_library/
    suds_1-3/
      eating/
      breathing/
      [other categories]/
    suds_4-6/
      eating/
      breathing/
      [other categories]/
    suds_7-9/
      eating/
      breathing/
      [other categories]/

Name files descriptively: eating_chewing_slow_01.mp3, breathing_nasal_close_02.mp3.
Include a simple text note in each folder: current SUDs rating + date rated.
Re-rate and update as the hierarchy shifts week over week.

STEP 3 — SOURCE 1: FOAMS DATABASE (START HERE)
The FOAMS (Functional and Affective Misophonia Sounds) database is a
peer-reviewed, pilot-rated misophonia sound set published on Zenodo.
32 files across 8 sound classes with distress ratings from a misophonia
sample — the only library specifically validated for this condition.

  1. Go to: https://zenodo.org
  2. Search: FOAMS misophonia
  3. Download the dataset (free, no account required)
  4. Sound classes include eating, breathing, sniffing, pen clicking,
     keyboard typing, and others — sort directly into hierarchy folders
     using the published distress ratings as a starting point, then
     adjust to your own response on first listen
  5. These are clean, calibrated recordings — use them as the backbone
     of the lower-to-mid hierarchy bands

STEP 4 — SOURCE 2: FREESOUND.ORG
For supplemental variation and any categories not covered by FOAMS.

  1. Go to: https://freesound.org (free account required)
  2. Search by trigger category: "chewing," "eating," "mouth breathing,"
     "sniffing," "nose breathing," "tapping," "clicking," etc.
  3. Filter: Duration 0:10–2:00, License = Creative Commons (any)
  4. Preview each before downloading — confirm it produces a recognizable
     response. If it produces nothing, it is not useful. If it produces
     a strong response, note its SUDs and file accordingly.
  5. Download as WAV or MP3 (WAV preferred for quality)
  6. Aim for variety within each category: different people, different
     intensities, different room acoustics

STEP 5 — SOURCE 3: YOUTUBE VIA yt-dlp
For sounds not available on Freesound — particularly close-proximity or
ASMR-style recordings that better simulate real exposure conditions.

  INSTALL yt-dlp (one-time):
    Via pip:     pip install yt-dlp
    Via Homebrew (Mac): brew install yt-dlp

  BASIC USAGE — extract audio only from a YouTube URL:
    yt-dlp -x --audio-format mp3 -o "%(title)s.%(ext)s" "YOUTUBE_URL"

  USEFUL SEARCH TERMS ON YOUTUBE:
    "eating sounds no music" / "chewing ASMR no talking"
    "mouth breathing sounds" / "nasal breathing close mic"
    "sniffing sounds" / "eating crunch sounds"
  
  Download only 30–90 second clips where possible. Long videos waste
  storage and most usable content is in the first 1–2 minutes.
  Trim to relevant segments using any audio editor (Audacity is free).

STEP 6 — SOURCE 4: SELF-RECORDING
The most ecologically valid source — sounds recorded in your actual
environment, from actual trigger sources. These are the highest-charge
recordings and belong in the upper SUDs bands.

  Use your phone's voice memo app (or any recording app).
  Position the microphone 1–3 feet from the sound source — closer
  than real life slightly inflates the charge, which is useful for
  building hierarchy headroom.
  Record 30–90 seconds per clip.
  Do this during Phase 1 when community meals and meetings are
  happening anyway — you're present, not conducting a session.
  Label immediately after recording before you forget the context.

STEP 7 — CALIBRATION PASS
Before beginning Phase 2, do one calibration pass through the library:
  - Put on headphones, lie down, reach entry state (SUDs 0–2)
  - Play each recording for 20–30 seconds
  - Note your actual SUDs response (not your predicted response)
  - Reassign to folders based on real response
  - Discard any recording that produces no response
  - Flag any recording that immediately floods (>8) — set aside for
    late Phase 3 or Phase 4
The calibrated library, not the sourced library, is what you work from.

MINIMUM VIABLE LIBRARY TO START CC:
  SUDs 1–3: 3+ recordings across 2+ trigger categories
  SUDs 4–6: 3+ recordings (you may not use these in Phase 2 but have
            them ready)
  SUDs 7–9: Have them filed but do not use them until Phase 3+

────────────────────────────────────────────────────────────────────────────────
B. PAIRED STIMULI PLAYLIST
────────────────────────────────────────────────────────────────────────────────

PURPOSE
Music that reliably produces genuine somatic positive emotion — physically
felt warmth, expansion, chills, welling, joy — when listened to with full
attention. This is the counterconditioning agent. Its quality determines
whether CC sessions write new associations or simply produce noise.

The music is not background. It is an active therapeutic input. If you are
not feeling it in your body, it is not doing its job.

CRITICAL RULE — PRESERVE POTENCY
Do not use this playlist for casual listening. Every casual play reduces
its emotional charge. Reserve it exclusively for CC sessions and the testing
process below. Treat it as a protocol tool, not a music library.

STEP 1 — CANDIDATE SELECTION
Identify 10–15 candidate tracks across a range of emotional textures.
Useful categories:

  Transcendent / awe-inducing:
    Orchestral, choral, cinematic scores that produce a sense of expansion
    or being moved by something larger than yourself.

  Warm / tender / connected:
    Music associated with genuine love, belonging, or deep safety. Can be
    personally meaningful — pieces tied to positive life moments, people,
    or places you love.

  Joyful / alive:
    Rhythmically engaging music that produces physical aliveness, not just
    pleasure. Not hype — genuine felt joy.

  Emotional release:
    Music that produces tears or welling — not sadness, but the kind of
    emotional fullness that releases something. This is a high-quality
    positive state for CC even if it does not feel conventionally positive.

Avoid: music associated with grief or loss, music you know too well to
feel anymore, high-energy music that produces alertness rather than felt
emotion, anything with lyrics that will pull cognitive attention away from
the somatic experience.

STEP 2 — SOMATIC TESTING (do this properly, not casually)
Test each candidate track individually:
  1. Lie down or sit fully upright. Headphones on.
  2. Close eyes. Take 5 slow breaths before pressing play.
  3. Give the track 2–3 minutes of full attention.
  4. Note: do you feel anything in your body? Where? Chest expansion?
     Warmth? Chills? Welling behind the eyes? Throat tightening in a
     good way? A sense of being held or lifted?
  5. Rate the somatic response on a simple 0–3 scale:
       0 = nothing felt, just heard
       1 = mild pleasant feeling, not clearly somatic
       2 = clear somatic response — warmth, expansion, or welling
       3 = strong, reliable somatic response within 2 minutes
  Keep only tracks scoring 2–3. Discard 0–1.

Do this across multiple sessions, not in a single sitting. Emotional
state affects response. A track that scores 3 on one day and 1 on another
is unreliable — set it aside.

STEP 3 — FINAL PLAYLIST ASSEMBLY
Select 3–5 tracks that:
  - Score 2–3 consistently across multiple testing sessions
  - Span different emotional textures (not all the same energy/tone)
  - Total 30–40 minutes combined
  - Feel like they belong together without jarring transitions

Order matters. Open with something that reliably produces felt emotion
quickly — this is your priming track. Subsequent tracks can build or
vary. The playlist should feel like it has an arc.

STEP 4 — DUAL-EAR SETUP TEST
Before the first CC session, test the full CC setup:
  Equipment: headphones with left/right channel control, or a
  splitter that allows independent volume control per ear.
  
  Method 1 — Software pan (simplest):
    Use any audio player (VLC, Audacity) that allows panning.
    Pan positive music to the dominant ear (typically right) at
    full volume. Pan trigger sound to non-dominant ear, reduce
    volume to approximately 30% of positive music level.
    Positive music should carry ~70% of auditory attention.
    
  Method 2 — Two devices:
    Phone in dominant ear playing positive playlist.
    Laptop or second device in non-dominant ear playing trigger sound.
    Independent volume control. Slightly more reliable than software pan.
    
  Test this setup without any trigger sound first — confirm the
  positive music is delivering felt emotion through one ear only.
  Then introduce a low-hierarchy trigger sound in the other ear at
  low volume. Adjust until the positive music remains dominant and
  the trigger is present but not overwhelming.
  
  This technical setup should feel completely natural before the
  first real CC session. The session is not the time to troubleshoot
  equipment.

STEP 5 — LABEL AND PROTECT
Save the playlist in a dedicated folder/app playlist labeled clearly
(e.g. "CC Positive" — nothing more descriptive than needed).
Do not share it, sync it to general playlists, or let it autoplay.
If it appears in shuffle, skip it.

READINESS CHECK BEFORE FIRST CC SESSION:
  □ Trigger library calibrated — at least 3 SUDs 1–3 recordings ready
  □ Positive playlist tested — consistent somatic response confirmed
  □ Dual-ear setup tested and working
  □ Entry state (SUDs 0–2 via resonance breathing) reliably reachable
  □ tVNS device (Nuropod) in hand and settings confirmed
